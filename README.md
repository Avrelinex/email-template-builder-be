## Description
Back end for email template builder.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run mongo db

```bash
docker run --name mongo -p 27017:27017 -d mongo
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Format

```bash
$ npm run format
```

## Lint

```bash
$ npm run lint
```
